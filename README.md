Denna Github Repository är en samlingspunkt för filer som användas för att onboarda Azure Lighthouse med fler olika RGs som ges olika typer av rättighter.
 
AzureLighthouseMultiRGTemplate.json är en template fil till Azure Lighthouse som används för att att onboarda fler olika RGs med olika rättighter
MultiRGParametersExample.json är ett exampel på hur parameter filen skall struktureras för att funka med Template
